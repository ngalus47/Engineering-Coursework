# C Programming for Engineers (EG1213)  

This folder contains labs and the semester-long project for C Programming. The course emphasized using **C** to solve engineering problems and apply concepts to hardware.  

## Semester Project: Autonomous Raspberry Pi Car  
- Programmed a Raspberry Pi-powered car to drive autonomously.  
- Integrated sensor data and applied control logic for navigation.  
- Showcased teamwork, debugging, and system integration.  

## Skills Gained  
- Proficiency in C programming for hardware interaction.  
- Hands-on experience with embedded systems and real-time decision making.
